# 911 模板语法

![[Hugo]]

## `.` dot 符号

代表上下文内容。

## 判断语句

### if 关键字

```Hugo
{{ if .Title }}{{ end }}
```

### with 关键字

检查其后跟着的变量是否存在，并将该变量设置为上下文。

```Hugo
{{ with .Title }}{{ end }}
```

### else 关键字

```Hugo
{{ with .Title }}
  ...
{{ else }}
  ...
{{ end }}
```
